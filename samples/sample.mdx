Copilot 早就不只是 IDE 里的补全，而是补全 + Chat + Agent Mode + Cloud Agent + CLI 五个层级的产品线。新手最容易把它和 Claude Code / Cursor 搞混。

IDE Agent Mode 写、Cloud Agent 跑长任务、PR 当验收线。三个层级各管一段，不要都用 IDE 里的 Agent 处理几十分钟的长任务。

完整教程：https://aiworkflowtutorials.com/docs/github-copilot
